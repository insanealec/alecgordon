# alecgordon

Personal website for Alec Gordon.

Started from the Vue 3 / Vite template.

## Deployment

Merge to main

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
